Web Development Certificate (Level 1)

The Web Development Certificate curriculum is designed to provide the skills necessary for an entry-level position as a Web Developer. Emphasis is upon the technical development of websites including networking, database concepts and web authoring using programming languages and software to produce interactive, and dynamic websites and web applications.

This is a level 1 certificate and is designed to be completed in 2 semesters for a full-time student.

Web Developer Specialist Certificate (Level 2)

The Web Developer Specialist Certificate curriculum trains the student to design and develop web pages and web business applications. It prepares the student to enter the workforce as a member of the company web development team for Internet/Intranet applications.

This is a level 2 certificate and is designed to be completed in 3 semesters for a full-time student.

Web Design and Development AAS

This degree is designed to provide students with technical and artistic skills required to use the web as an interactive communications medium. Students pursuing this degree will learn basic and advanced page design skills, digital imaging techniques suitable for the web, basic client- and server-side programming methods (including form validation and shopping carts), problem-solving skills, interpersonal workgroup skills, and techniques for delivering video and audio over the Internet.

The Web Production and Design Associate of Applied Science Degree recently received Texas Skill Standards Board (TSSB) Program Recognition for integrating the Web Development and Administration Skill Standards into the Brookhaven curriculum. Program graduates will be prepared with the skills and knowledge, as defined by the industry, for employees to be successful in the workplace. For more information about the TSSB program, go to the Texas Skill Standards Repository on the TSSB’s website(www.tssb.org).

This is an Associate in Applied Science Degree and is designed to be completed in 2 years for a full-time student.
