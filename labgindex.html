The Web Production and Design Associate of Applied Science Degree recently received Texas Skill Standards Board (TSSB) Program Recognition for integrating the Web Development and Administration Skill Standards, as defined by the industry, into the curriculum. For more information about TSSB, visit the TSSB website: http://www.tssb.org.

Web design and development is a fast growing and diverse career field. Careers can range from graphic design to programming to server administration just to mention a few. The Web Design and Development Program at Brookhaven is designed to give you a solid background in each of these areas, while also allowing you to specialize in courses specific to your interests.

At Brookhaven, we are constantly updating the courses and skills taught to keep pace with this rapidly evolving field to ensure our students have the skills most in demand from industry including the following:

  • HTML5
  • CSS3
  • Photoshop
  • Dreamweaver
  • WordPress
  • JavaScript
  • MySQL
  • PHP
  • ASP.NET
  • Mobile devices
  • Responsive design
  • SEO

 

Students begin by learning basic and advanced page design and digital imaging techniques, then progress to more advanced skills such as client and server programming for form validation, user interaction, database driven websites and shopping carts. Students will learn skills for problem solving and working within groups. A capstone project for the AAS degree includes the development of a complete website for a local non-profit agency. Our computer labs include the latest software and development tools to ensure you have the skills in high demand from industry.

Careers in Web Design and Development

This career path can lead to a wide variety of job positions, including:

  • Web Designer
  • Web Developer
  • Web Programmer
  • Webmaster/Web administrator
  • Web graphics designer
  • Digital media specialist
  • Web software engineer
  • Systems analyst
  • Database developer

 

Web skills can even lead to jobs in various other disciplines that require Web skills, including:

  • Technical writer
  • Technical support
  • Information architect
  • Product or program manager
  • Graphic designer
  • Layout artist
  • Digital developer
